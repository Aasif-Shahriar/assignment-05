What is the difference between innerHTML, innerText, and textContent ?

Question er properties gulo hocche DOM-er properties. Egulo use kore dynamically HTML elements and content update kora jay.

1. innerHTML diye HTML tag and content 2ta eksathe access kora jay.

2. innerText use kore ekta element shudhu content ke access kora hoy and ei property css diye hide kora content access korte pare na.

3. textContent property shudhu matro element er content er upor focus kore kono dhoroner css style er upor focus kore na. Eta hide kora text ke-o access korte pare.

What is event delegation in the DOM?

What is event bubbling in the DOM?

Jokhon ekta element er upor eventListener add kora hoy tokhon tar sathe ancestor gulor o event active hoye jay. Eta bubbling hote hote tar previous element ke trigger korte thake. So eta stop korar ekta way ache ar seta holo stopPropagation() method use kora.
